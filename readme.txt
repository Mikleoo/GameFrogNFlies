
Bienvenue dans le projet FrogNFlies !

Ce projet est un jeu web dans lequel une grenouille doit se déplacer dans une grille pour aller manger les mouches avec le moins de déplacement possible. C'est un jeu réalisé en équipe de 2 en 4 jours. Le but étant de s'exercer à typescript, le projet n'est pas parfait que ce soit au niveau de css, de l'organisation du code ou de la sécurité...

J'ai quand même mit tout les fichiers du jeu avec la base de donnée dans ce repository pour pouvoir voir un peu le code et pouvoir le tester potentiellement en local.
